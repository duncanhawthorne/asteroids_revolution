# Pacman ROLL
Pacman ROLL is Pacman plus physics.

* Rotate the maze to move Pacman and the ghosts.
* Eat all the dots and avoid the ghosts. 
* Play for speed to top the leaderboard. 
* Play Pacman ROLL at https://pacman-roll.web.app
